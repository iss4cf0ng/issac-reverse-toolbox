# issac-reverse-toolbox

Personal reverse engineering toolbox with custom tools, scripts, and malware analysis notes.

This repository also serves as my personal toolbox while setting up my experimental environment.

⭐ If you find this toolbox useful, give it a star — it motivates me to add more tools, notes, and examples regularly!

## Disclaimer
All tools, scripts, and notes are for **educational and research purposes only**.  
Do **not use them for illegal activities**. Use at your own responsibility.

Some tools are **NOT** my work. Please use them only in your personal virtual machines.

---

## Samples (For Analysis / Learning)

This repository also contains **analyzed or unpacked samples** for educational and research purposes.  
- Do **not run these samples on your main system**. Use a properly isolated virtual machine.  
- Each sample may include notes, scripts, or extracted binaries to study behavior safely.
- To prevent unintentional execution, all samples are distributed as password-protected ZIP archives (`.zip`) with the password `infected`.
